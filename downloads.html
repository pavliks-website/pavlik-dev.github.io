Peachy for Chrome/Firefox

A peachy theme with clouds because I love peach color.
Supports ancient and latest Chrome versions, and also Firefox.
(click for full size)

To install, first download the theme as a file by right-clicking the version you want and selecting "Save link as...".
Then go to chrome://extensions, enable Developer mode and drag and drop .crx file into the browser window.
